# Student Mental Health Analysis Using SQL

## 📌 Project Overview
This project analyzes factors affecting students' mental health using SQL. 
The objective is to identify high-risk students and understand key contributing factors.

## 🗂 Dataset Features
- Academic Pressure
- Financial Concerns
- Depression
- Anxiety
- Isolation
- Future Insecurity
- Sleep Patterns
- Social Relationships

## 🧹 Data Cleaning
- Removed duplicate records
- Standardized gender values
- Validated scale columns (1–5 range)
- Removed invalid age entries
- Trimmed text fields

## 📊 Key Insights
- Higher academic pressure increases depression levels.
- Financial concerns significantly impact anxiety.
- Students with poor sleep show higher stress.
- Risk score model identified high-risk students.

## 🧮 Risk Score Formula
risk_score = academic_pressure + financial_concerns + depression + anxiety + isolation + future_insecurity
